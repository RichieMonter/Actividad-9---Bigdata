Instrucciones rápidas:

1. Ajusta variables de conexión MySQL en los notebooks (usuario, contraseña, host, puerto).
2. Coloca worldcities.csv en la misma carpeta si deseas usar datos reales.
3. Ejecuta CollectWeather.ipynb y Collect_flights.ipynb en Jupyter.
4. Usa gans.sql para crear las tablas si prefieres hacerlo manualmente.
5. Sube los archivos modificados a tu repositorio GitHub y commit/push.
